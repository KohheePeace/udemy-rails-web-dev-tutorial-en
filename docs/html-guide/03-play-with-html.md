## Let's add some new HTML tag
`test.html`
```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>

<body>
<h1>Here is a title.</h1>
<h2>Here is a subtitle.</h2>
<ul>
  <li>Coffee</li>
  <li>Tea</li>
  <li>Milk</li>
</ul>
</body>

</html>
```
