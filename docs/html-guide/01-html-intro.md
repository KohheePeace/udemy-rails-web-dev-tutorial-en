## 
`terminal`
```
mkdir html-test
cd html-test
touch test.html
```

Copy the W3 schools example and play with it.

`test.html`
```html
<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8">
<title>Title of the document</title>
</head>

<body>
Content of the document......
</body>

</html>
```

Open this html file in chrome (**web browser**).

`terminal`
```bash
# under html-playground folder
open test.html
```
